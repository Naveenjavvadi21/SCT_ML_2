# 🛍️ Customer Segmentation using K-Means Clustering

## 📌 SkillCraft Technology - Machine Learning Internship

### Task 2: Customer Segmentation using K-Means Clustering

## 📖 Project Overview

This project was developed as part of the **SkillCraft Technology Machine Learning Internship – Task 2**. The objective is to segment customers into different groups based on their purchasing behavior using the **K-Means Clustering** algorithm.

Customer segmentation enables businesses to better understand customer behavior, improve marketing strategies, and provide personalized services.

---

## 🎯 Objectives

- Perform customer segmentation using K-Means Clustering.
- Analyze customer purchasing behavior.
- Determine the optimal number of clusters using the Elbow Method.
- Visualize customer groups.
- Gain business insights from clustering results.

---

## 📂 Dataset

**Dataset:** Mall Customer Segmentation Dataset

### Features

- Customer ID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1–100)

### Features Used for Clustering

- Annual Income (k$)
- Spending Score (1–100)

## 📊 Project Workflow

1. Import required libraries.
2. Load the dataset.
3. Explore and preprocess the data.
4. Select important features.
5. Use the Elbow Method to determine the optimal number of clusters.
6. Train the K-Means model.
7. Predict cluster labels.
8. Visualize customer segments.
9. Interpret the clustering results.

---

## 📈 Results

The K-Means model successfully grouped customers into distinct clusters based on their annual income and spending score. These customer segments can help businesses:

- Identify high-value customers.
- Design targeted marketing campaigns.
- Improve customer retention.
- Enhance business decision-making.
